Konfluenz

Production and Performance Mixer

16 HP | +12: n mA, -12: n mA

Konfluenz is a high-quality three chanel eurorack mixer designed for both studio production and live performance. Each chanel features a solo and mute button, as well as a send and return for external effects.

Features

Solo/Cue | Mute

Mute

When the mute button is pressed, the channel is muted. The mute button will light up yellow when pressed.

Solo/Cue

When the Solo/Cue button is pressed, the channel is either soloed or cued, if Cue mode is enabled. The Solo/Cue button will light up blue when pressed. Both Solo and Cue override the Mute button.

Send - Return

Send - Return

Each chanel features a send and return for external effects. The send and return are both 3.5mm jacks. The send is pre-, and the return post-fader. The Mix knob controls the amount of the return signal that is mixed with the channel signal.

Output

Headphone Out

The Headphone Out gives you a way to monitor your mix. The Mix knob lets you determine the amount of main and cue signal that is sent to the Headphone Out. The Headphone Out is mono.

Main Out

The Main Out gets your mix ready to be sent to your speakers or another module. It is controlled by the main fader.
